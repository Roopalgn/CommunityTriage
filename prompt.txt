Elevate my UI from a clean dashboard to a high-signal, decision-focused interface.

Core goal:
Make the UI guide user decisions instantly with minimal cognitive load.

Improvements:

1. Visual hierarchy:
   - Ensure only ONE dominant area per screen
   - Make decision panels higher contrast than background content

2. Add semantic color system:
   - Encode urgency (critical/high/medium) into backgrounds, not just badges
   - Use subtle gradients to reinforce severity

3. Improve scanability:
   - Add left border indicators for priority
   - Increase visual weight of top-ranked items
   - Reduce text density in lists

4. Reduce unused space:
   - Compress hero/mission section into a compact action bar
   - Prioritize actionable UI over descriptive UI

5. Enhance metrics:
   - Emphasize numbers with size and color
   - Add hover states and micro-interactions

6. Strengthen decision UI:
   - Highlight recommended actions (e.g., best volunteer)
   - Make primary actions visually dominant
   - Reduce friction for obvious decisions

7. Improve motion:
   - Add subtle transitions for hover, selection, and navigation
   - Animate key feedback (assignment, selection)

8. Upgrade forms:
   - Stronger focus states
   - Better input affordance

9. Add interpretation layer:
   - Convert raw metrics into insights (e.g., “High load detected”)

10. Maintain constraints:
   - Minimal, modern design
   - No heavy glow or excessive blur
   - Performance-first

Output:
- Updated component styles
- Refined layout hierarchy
- Improved interaction states

Fix vertical layout issues across all pages:

- Ensure app uses full viewport height (min-height: 100vh)
- Convert layout to flex column structure
- Make main content area flex: 1 so it fills remaining space
- Ensure page sections expand vertically instead of collapsing
- Use grid/flex layouts so side-by-side panels stretch to full height
- Add overflow handling (scroll inside panels, not entire page)
- Eliminate dead space at bottom of pages without using fixed heights

Goal:
Every screen should feel full, balanced, and vertically complete with no empty voids.
